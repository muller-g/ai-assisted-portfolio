## 📱 Responsividade (OBRIGATÓRIO)

O site deve funcionar perfeitamente em:

- Mobile (prioridade máxima)
- Tablet

Regras:
- Mobile-first
- Layout fluido (sem quebras)
- Tipografia adaptativa
- Testar em múltiplas resoluções

---

## ⚡ Performance (CRÍTICO)

- Utilizar SSR/SSG do Next.js quando possível
- Evitar renderizações desnecessárias
- Lazy loading de imagens e componentes
- Otimizar bundle size
- Usar `next/image`

---

## 🔍 SEO (EXTREMAMENTE IMPORTANTE)

Toda página deve ser otimizada para SEO:

### Regras obrigatórias:
- Uso correto de `<title>` e `<meta description>`
- Estrutura semântica (h1, h2, h3…)
- URLs amigáveis
- Open Graph (Facebook/LinkedIn)
- Meta tags para Twitter
- Sitemap.xml
- Robots.txt

### Conteúdo:
- Textos claros e com palavras-chave (ex: "Desenvolvedor Fullstack", "Laravel", "Next.js")
- Evitar conteúdo vazio ou genérico

### Performance SEO:
- Lighthouse alto (90+)
- Core Web Vitals otimizados

### Palavras-chave estratégicas (OBRIGATÓRIO)

O conteúdo deve incluir variações de:

- Desenvolvedor Fullstack
- Desenvolvedor com IA
- Desenvolvedor de agentes de IA
- Automação com inteligência artificial
- Integração com OpenAI
- Laravel + IA
- Next.js + IA

Evitar stuffing, mas garantir presença natural nos textos.
