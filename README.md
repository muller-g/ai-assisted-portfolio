## Instruções para agentes IA

1. **Leia o [AGENTS.md](./AGENTS.md) primeiro** — ele define as regras que nunca podem ser ignoradas
2. **Consulte o [action-plan.md](./action-plan.md)** — para entender o estado atual e o que precisa ser feito
3. **Atualize o [timeline.md](./timeline.md)** — toda vez que implementar ou alterar algo no código
4. **Responda sempre em português (PT-BR)**
5. **Destaque IA e Automação** como diferencial do desenvolvedor
